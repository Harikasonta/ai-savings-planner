# 💡 AI-Based Smart Savings Planner

A full-stack web application that helps users **track their income and expenses** and provides a **personalized savings plan using intelligent analysis**.

This project is designed to solve a real-world problem — managing money effectively — by combining **financial tracking + AI-based planning**.

## 🎯 Features

### 🔐 Authentication

* User Signup & Login
* Secure password hashing using bcrypt
* JWT-based authentication

---

### 💰 Financial Tracking

* Add income from multiple sources
* Add expenses with categories (food, transport, etc.)
* Dynamic calculation of:

  * Total income
  * Total expenses
  * Savings

---

### 🤖 AI-Based Financial Planner

Instead of just showing data, the system provides:

* 📊 Budget breakdown (Needs, Wants, Savings)
* 💡 Personalized saving recommendations
* 📉 Spending analysis
* 🧠 Actionable financial plan

Example:

* Reduce spending in high categories
* Maintain a minimum saving percentage
* Balance lifestyle vs savings

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/ai-savings-planner.git
cd ai-savings-planner
```

---

### 2. Setup Backend

```
cd backend
npm install
npm run dev
```

Create a `.env` file:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```

---

### 3. Setup Frontend

```
cd frontend
npm install
npm start
```

## 🧠 How It Works

1. User registers and logs in
2. Adds income and expenses
3. Data is stored in MongoDB
4. Backend analyzes financial data
5. AI module generates:

   * Savings plan
   * Budget allocation
   * Actionable suggestions
